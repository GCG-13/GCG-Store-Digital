# GCG Store - Digital Marketplace 🛒

Plataforma de comercio electrónico diseñada para la venta de artículos digitales y bienes de juegos. Este proyecto demuestra la capacidad de gestionar catálogos dinámicos y usuarios en tiempo real.

## 🛠️ Tecnologías Utilizadas
- **Frontend:** HTML5, CSS3, JavaScript (ES6+).
- **Backend as a Service (BaaS):** Firebase (Hosting, Firestore para inventario).
- **Mobile Experience:** Configurado como **PWA (Progressive Web App)** para acceso rápido desde dispositivos móviles.

## 🚀 Funcionalidades Clave
- **Gestión de Inventario:** Listado dinámico de productos desde la base de datos.
- **Sistema de Cuentas:** Registro y manejo de perfiles de usuario.
- **Optimización:** Carga rápida y diseño responsivo para gamers.

## 🛡️ Nota sobre el módulo de pagos
El sistema de checkout y pasarela de pagos se mantuvo en **modo de prueba (Sandbox/Desarrollo)**. Por razones de seguridad y cumplimiento de normativas de manejo de datos financieros, la integración final de producción no ha sido desplegada en este repositorio público para proteger las credenciales y la integridad del sistema.

## 📉 Estado del Proyecto
Actualmente en fase de **MVP (Producto Mínimo Viable)**. El desarrollo se enfocó en la arquitectura de datos y la experiencia de usuario (UX).
